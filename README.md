# WhatsApp-to-CardDav

Save the WhatsApp profile pictures of your friends to their CardDAV contacts (e.g. on Nextcloud, Google and Apple contacts).

## Getting started

```sh
npm i
cp sample.env .env
# add your own values to .env
npm start
```
